## 🚀 התקנה והפעלה

### הפעלה בסיסית
1. פתחו את הקובץ `index.html` בדפדפן (Chrome/Edge מומלצים).
2. זהו — המערכת מוכנה לשימוש.

### התקנה כאפליקציה (PWA)
המערכת תומכת בהתקנה כאפליקציה עצמאית:
- **Chrome**: לחצו על אייקון ההתקנה (⊕) שמופיע בשורת הכתובת, או: תפריט ← "התקן את SunDay"
- **Edge**: תפריט ← "אפליקציות" ← "התקן אתר זה כאפליקציה"
- **מכשיר נייד**: "הוסף למסך הבית" מתפריט הדפדפן

> **חשוב:** כל הנתונים נשמרים ב-localStorage של הדפדפן. אם תמחקו את נתוני הדפדפן — הנתונים יימחקו. גבו באופן קבוע!

## ⚙️ הגדרות ראשוניות (חובה!)

לפני שמתחילים — הכניסו את ההגדרות הבסיסיות. עברו ל**הגדרות** מתפריט הניווט.

### 1. הגדרת תאריכי משימה (כלליות ← משימה כללית)
- **תאריך התחלת משימה** — היום הראשון של המילואים
- **תאריך סיום משימה** — היום האחרון
- לחצו **💾 שמור**

> תאריכים אלו קובעים את טווח כל המערכת. ניתן בנוסף להגדיר טווחים ספציפיים למודול הנוכחות ולמודול השיבוץ בנפרד.

### 2. הגדרת מבנה פלוגה (כלליות ← מבנה פלוגה)
- בחרו כמה מחלקות יש (1–6)
- תנו שם לכל מחלקה (למשל: "מחלקה א'", "מחלקה ב'")
- לחצו **💾 שמור מבנה**

### 3. הגדרות נוכחות (נוכחות)
- **ספי נוכחות**: הגדירו אחוז קריטי (אדום) ואחוז חריג (צהוב) — למשל 50% ו-60%
- **כללי ספירה**:
  - האם יום יציאה נספר כחיסור?
  - האם יום חזרה נספר כחיסור?
  - שישי-שבת נספרים כיום/יומיים?
- **שעות ברירת מחדל**:
  - שעת יציאה טיפוסית (למשל 14:00)
  - שעת חזרה טיפוסית (למשל 17:00)
  - סף יציאה — אם חייל יוצא אחרי שעה זו, הוא עדיין נספר כנוכח באותו יום
- **מינימום מפקדים** למחלקה
- **בעלי תפקידים** — הגדירו תפקידים נדרשים (חובש, נהג, רס"פ וכו'), ברמת מחלקה או פלוגה, עם מינימום נדרש

### 4. הגדרות שיבוץ (שיבוץ)
- **שעות מנוחה מינימליות** — כמה שעות מנוחה נדרשות בין משמרות (למשל 8)
- **ימים ברירת מחדל** — כמה ימים לשבץ בברירת מחדל (למשל 7)

## ☀️ מודול נוכחות

כאן מנהלים את כל החיילים, ההיעדרויות, והנוכחות היומית.

### לוח בקרה פלוגתי
הטאב הראשון ("📊 לוח בקרה פלוגתי") מציג:
- **סיכום נוכחות פלוגתי** — כמה נוכחים/נעדרים בכל מחלקה
- **טבלת סיכום מחלקתי** — עם אחוזי נוכחות ומצב מפקדים
- **כרטיסי מחלקות** — כרטיס לכל מחלקה עם רשימת הנעדרים

ניתן לבחור טווח תאריכים לתצוגה או לחזור ל"כל המשימה".

### ניהול חיילים (טאב מחלקתי)
לכל מחלקה יש טאב נפרד. בתוך כל טאב:

#### הוספת חיילים
- **הוספה בודדת**: לחצו ➕ הוסף חייל — מלאו שם, טלפון, דרגה, מחלקה, מפקד (חייל/מ"כ/מפקד), ותפקידים
- **הוספה קבוצתית**: לחצו 📋 הוספה מרובה — הזינו שמות (כל שם בשורה חדשה), בחרו מחלקה, דרגה קבועה, ומפקד (חייל/מ"כ/מפקד). לחצו "💾 שמור"
- **ייבוא מאקסל**: לחצו 📂 ייבוא — העלו קובץ אקסל עם עמודות: שם, טלפון, דרגה, מחלקה, תפקידים, מפקד

#### עריכת חייל
לחצו על שם החייל בטבלה — ייפתח דיאלוג עריכה בו תוכלו לעדכן את כל הפרטים.

#### שדות חייל
| שדה | הסבר |
|---|---|
| שם | שם מלא (חובה) |
| טלפון | מספר ליצירת קשר |
| דרגה | דרגה צבאית |
| מפקד | חייל רגיל / מ"כ / מפקד |
| מחלקה | שיוך למחלקה |
| תפקידים | תפקידים חיוניים (חובש, נהג, רס"פ וכו') |
| פעיל | האם החייל פעיל או מושבת |

### ניהול היעדרויות
בתוך כל טאב מחלקתי, לחצו על ➕ בצד שם החייל או השתמשו בטאב "🏖️ היעדרויות":

#### הוספת היעדרות
- **חייל** — בחרו חייל
- **סיבה** — חופשה / קורס / מחלה / מיוחד / אחר
- **תאריך התחלה** + **שעת יציאה**
- **תאריך סיום** + **שעת חזרה**
- **הערה** (אופציונלי)

> **טיפ:** סיבת ההיעדרות משפיעה על הצגת החייל בדוחות (חולים מופרדים, יוצאים/חוזרים היום נספרים בנפרד).

### Heatmap — מפת חום נוכחות
בכל טאב מחלקתי, לחצו על טאב "🗓️ Heatmap":
- **שורות** = חיילים
- **עמודות** = ימים
- **צבעים**: ירוק = נוכח, אדום = חסר, כתום = חופשה, כחול = קורס, סגול = מיוחד, אפור = עתידי
- בשורה התחתונה: אחוז נוכחות יומי + סטטוס מפקדים

### לוח שנה (Calendar)
טאב "📅 לוח שנה" — תצוגת חודש עם אחוזי נוכחות יומיים בכל תא, ימים קריטיים מסומנים באדום.

## 🔫 מודול שיבוץ

ניהול עמדות שמירה ושיבוץ אוטומטי של חיילים למשמרות.

### טאב חיילים (👥)
מציג את כל החיילים הזמינים לשיבוץ, מחולקים לטאבים לפי מחלקה.
- **🔄 סנכרן מנוכחות** — מייבא/מעדכן חיילים ממודול הנוכחות (כולל שם, מחלקה, תפקידים, דרגת פיקוד)
- לכל חייל ניתן להגדיר:
  - **ימים חסומים** — ימים בשבוע שבהם אין לשבץ אותו
  - **תאריכים חסומים** — תאריכים ספציפיים
  - **עמדות מועדפות** — למתן עדיפות באלגוריתם

> **חשוב:** חיילי השיבוץ מסונכרנים ממודול הנוכחות. הוסיפו חיילים קודם בנוכחות, ופה סנכרנו.

### טאב עמדות (📍)

#### הוספת עמדה
לחצו ➕ הוסף עמדה:

| שדה | הסבר |
|---|---|
| שם עמדה | שם ייחודי (למשל: "שער ראשי") |
| שיוך | מחלקתית (שייכת למחלקה אחת) / פלוגתית (כולם) |
| מחלקה | אם מחלקתית — לאיזו מחלקה |
| רוטציה | אם פלוגתית — ניתן להגדיר רוטציה לפי שעות/ימים/תאריכים |
| מספר חיילים | כמה חיילים נדרשים בכל משמרת |
| אורך משמרת | אורך המשמרת בשעות (למשל 4) |
| שעות פעילות | מ-שעה עד שעה (למשל 0–24 ל-24/7, או 6–22) |
| ימים פעילים | באילו ימים העמדה פעילה |
| עדיפות | 0 = רגילה, מספר גבוה = עדיפות גבוהה יותר |
| דרישת מפקד | אין / מפקד / סמל מחלקה — האם העמדה דורשת חייל בדרגת פיקוד + כמה מפקדים נדרשים |
| תפקיד נדרש | אופציונלי — תפקיד מתוך רשימת בעלי התפקידים שהוגדרו בהגדרות הנוכחות (למשל: נהג, חובש) + כמה בעלי תפקיד נדרשים |

#### ייבוא/ייצוא עמדות
- **📂 ייבוא** — ייבוא מקובץ אקסל
- **📊 ייצוא** — ייצוא כל העמדות לאקסל

### טאב שיבוץ (📋)

#### יצירת שיבוץ
1. בחרו **תאריך התחלה**
2. הגדירו **כמה ימים** לשבץ
3. לחצו **🔄 צור שיבוץ**

האלגוריתם ישבץ אוטומטית את החיילים לעמדות בהתבסס על:
- **זמינות** — לא ישובץ חייל שיש לו היעדרות, יום חסום, או שלא נח מספיק
- **איזון עומס** — מחלק שעות שוות ככל האפשר בין החיילים
- **מנוחה מינימלית** — שומר על מרווח מנוחה בין משמרות
- **גיוון עמדות** — מעדיף לשבץ חיילים לעמדות שונות
- **עדיפות עמדות מועדפות** — נותן בונוס לעמדות שהחייל העדיף
- **דרישת מפקד** — אם עמדה דורשת מפקד, ישובץ בעדיפות חייל בדרגת פיקוד מתאימה
- **תפקיד נדרש** — אם לעמדה הוגדר תפקיד נדרש (📌), ישובץ בעדיפות חייל שמחזיק בתפקיד זה

#### תצוגת השיבוץ
- **טבלה** — שורות = משמרות (שעה × תאריך), עמודות = עמדות + "בבית" + "במנוחה"
- **עמודת "בבית"** — חיילים שאינם זמינים (חופשה, מחלה וכו'), עם סיבת ההיעדרות בצבע
- **עמודת "במנוחה"** — חיילים זמינים שאינם משובצים כרגע
- **נעילת תאים** — לחצו על תא בשיבוץ כדי לנעול/לפתוח אותו (🔒). תא נעול לא ישתנה ביצירת שיבוץ חוזרת
- **הדגשת חייל** — בחרו חייל מהרשימה לראות את כל המשמרות שלו מודגשות
- **גרירה (Drag & Drop)** — ניתן לגרור חיילים בין תאים בלוח השיבוץ

#### אזהרות
המערכת מציגה אזהרות אוטומטיות:
- ⚠️ חייל ששובץ בזמן שאינו זמין (חופשה/יום חסום)
- ⚠️ חייל שלא קיבל מנוחה מספיקה
- ⚠️ עמדה שדורשת מפקד אבל שובצו אליה חיילים ללא דרגת פיקוד
- 📌 עמדה שדורשת תפקיד מסוים (למשל נהג) אבל אף חייל משובץ אינו בעל התפקיד
- ⚠️ חלוקת שעות לא שוויונית

### טאב עומס (⚖️)
מציג סטטיסטיקות עומס שמירות:
- **שעות כל חייל** — כמה שעות שמירה שובצו לו
- **מספר משמרות**
- **חריגים** — חיילים עם עומס גבוה/נמוך במיוחד
- התצוגה מחולקת לטאבים לפי מחלקות

## 📊 לוח בקרה (דשבורד)

מבט-על מהיר על מצב הפלוגה ביום נבחר.

### רכיבים:
1. **כרטיסי סיכום** — תאריך, אחוז נוכחות כללי, סה"כ חיילים, נוכחים, נעדרים
2. **טבלת נוכחות** — שורה לכל מחלקה עם: סה"כ, נוכחים, נעדרים, אחוז, סטטוס מפקדים
3. **כרטיסי מחלקות** — כרטיס ויזואלי לכל מחלקה עם רשימת נעדרים
4. **שיבוץ יומי** — אם יש שיבוץ ליום — מציג טבלת משמרות מקופלת (לחצו לפתיחה)

### שימוש:
- בחרו תאריך בחלק העליון
- לחצו **"היום"** לחזרה ליום הנוכחי
- הכל מתעדכן אוטומטית

## 📄 דוחות וייצוא

### דוח נוכחות (בפורמט וואטסאפ)
עברו לטאב **דוחות** ←

#### דוח מהיר
- **📋 העתק דוח מהיר (היום)** — מעתיק ישירות ללוח (clipboard) דוח נוכחות של היום, מוכן להדבקה בוואטסאפ
- **📥 הורד דוח היום (TXT)** — מוריד קובץ טקסט מעוצב

#### דוח מותאם (📄 צור דוח מותאם)
ניתן לבחור:
- **שם פלוגה** — למשל "פלוגה ג'"
- **תאריך / טווח** — יום בודד או טווח ימים
- **מחלקות** — כולן או ספציפיות
- **הגדרות ספירה**:
  - יוצא היום נספר כ-נוכח/חסר
  - חוזר היום נספר כ-נוכח/חסר
- **תצוגה מקדימה** — צפייה בפורמט וואטסאפ או טקסט מעוצב לפני השליחה

#### מבנה הדוח
הדוח מחולק לפי מחלקות, ובכל מחלקה:
- ✅ נוכחים (רשימה)
- ❌ חסרים (רשימה)
- 🤒 חולים (רשימה)
- 🏠 יוצאים היום (רשימה)
- 🔙 חוזרים היום (רשימה)
- סה"כ נוכחים/מתוך סה"כ

### ייצוא אקסל

#### ממודול הנוכחות
- **דוח חייל בודד** — מ-DetailsCard של חייל
- **דוח מחלקה** — מטאב המחלקה
- **דוח פלוגתי** — מלוח הבקרה הפלוגתי
- **ייצוא כל הנוכחות (Excel)** — מהגדרות ← ניהול נתונים: קובץ אקסל עם כל המחלקות, החיילים, וההיעדרויות

#### ממודול השיבוץ
- **📊 ייצוא שיבוץ** — טבלת שיבוץ מלאה לאקסל (תאריך × שעה × עמדות)
- **📊 ייצוא חיילים** — רשימת חיילי השיבוץ
- **📊 ייצוא עמדות** — רשימת העמדות
- **דוח חייל** — מכרטיס חייל: ציר זמנים עם כל המשמרות והמנוחות

## ❓ שאלות נפוצות

### איך מתחילים ממש מאפס?
1. **הגדרות** → הגדירו תאריכי משימה ומבנה פלוגה
2. **נוכחות** → הוסיפו חיילים (בודדים, קבוצתיים, או ייבוא מאקסל)
3. **נוכחות** → הוסיפו היעדרויות ידועות
4. **שיבוץ** → סנכרנו חיילים מנוכחות (🔄)
5. **שיבוץ** → הוסיפו עמדות
6. **שיבוץ** → צרו שיבוץ (🔄 צור שיבוץ)
7. **דוחות** → ייצאו דוח נוכחות לוואטסאפ

### חייל לא מופיע בשיבוץ?
- ודאו שהחייל **פעיל** בנוכחות
- ודאו שביצעתם **סנכרון מנוכחות** בטאב חיילים של השיבוץ
- בדקו שהחייל לא חסום ליום/תאריך הרלוונטי
- בדקו שאין לו היעדרות באותם תאריכים

### שיבוץ מראה אזהרות — מה לעשות?
- קראו את פירוט האזהרות (מופיעות מעל הטבלה)
- אפשר לנעול תאים (🔒) ולהריץ שיבוץ חוזר — תאים נעולים יישמרו
- אפשר לגרור חיילים בין תאים ידנית

### מה זה שדה "מפקד"?
שדה המפקד בחייל מגדיר את דרגתו הפיקודית:
- **מפקד** — מפקד מחלקה, נותן עדיפות גבוהה מאוד לעמדות שדורשות מפקד
- **מ"כ** — מפקד זוטר / סמל מחלקה, נותן עדיפות גבוהה לעמדות שדורשות מפקד
- **חייל רגיל** — ללא עדיפות מפקדית

> **שימו לב:** דרישת מפקד בעמדה מאפשרת לקבוע **כמה** מפקדים נדרשים (ברירת מחדל: 1). האלגוריתם ישבץ בעדיפות חיילים בדרגת פיקוד, ואם לא מספיק משובצים — תוצג אזהרה עם הכמות החסרה.

### מה זה "תפקיד נדרש" בעמדה?
ניתן לקבוע שעמדה דורשת תפקיד מסוים — מתוך רשימת בעלי התפקידים שהוגדרו בהגדרות הנוכחות (למשל: נהג, חובש, קשר).
- ניתן להגדיר **כמה** בעלי תפקיד נדרשים (ברירת מחדל: 1)
- העמדה תסומן ב-📌 בראש העמודה בשיבוץ (עם כמות אם > 1)
- האלגוריתם ייתן עדיפות לחיילים שמחזיקים בתפקיד הנדרש
- אם לא מספיק בעלי תפקיד משובצים — תוצג אזהרה עם הכמות החסרה

### איך מגבים נתונים לפני שמוחקים דפדפן?
הגדרות ← כלליות ← **גיבוי כל המערכת (JSON)** — מוריד קובץ שניתן לשחזר בכל עת.

### איך מעבירים נתונים למחשב אחר?
1. במחשב המקור: הגדרות ← **📤 ייצוא נוכחות (Excel)** ו/או **גיבוי JSON**
2. העבירו את הקובץ
3. במחשב היעד: הגדרות ← **📥 ייבוא** את הקובץ המתאים

### הנתונים נמחקו! אפשר לשחזר?
אם יש לכם קובץ גיבוי (JSON/Excel) — כן, מהגדרות ← שחזור. אם אין — לצערנו לא.
